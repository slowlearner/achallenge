
## Installation

```bash
$ npm install
```

## Running the app

```bash
# start docker
$ docker-compose up -d
# copy the sample configuration
# and modify the values
$ cp .env.sample .env
# development
$ npm run start

# watch mode
$ npm run start:dev

```


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
